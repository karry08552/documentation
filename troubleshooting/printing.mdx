<Accordion title="Printer exe Installation Verification">
Make sure that print client exe has been installed from portal.
</Accordion>

<Accordion title="Local Printer Online Status Verification">

The printer must be connected to the system for printing functionality.
</Accordion>

<Accordion title="Default Printer Configuration Conflict">
When utilizing the TS printer default option, ensure that the local printer is also designated as the default.
</Accordion>
