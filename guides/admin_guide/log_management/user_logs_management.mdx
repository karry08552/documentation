## Introduction

User Logs offer a detailed view of each user's activities, providing critical information to track user behavior and status.

### User Logs Key Features

* <b>Current Status: </b> View the real-time status of users, whether they are currently logged in or logged out.
* <b>Login and Logout Times:</b> Access detailed timestamps of when users log in and log out, ensuring precise tracking of user activity.
* <b>Email IDs:</b> Retrieve user email addresses associated with their login sessions for easy identification and communication.
* <b>Session Type:</b>  Identify the type of session, such as remote login or HTML login, for each user activity.

* <b>Session ID:</b> Retrieve unique session IDs associated with each user session, facilitating session management and troubleshooting.
